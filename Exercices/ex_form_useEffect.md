# Exercice 

Créez un formulaire avec un seul champ obligatoire, par exemple `name`

1. Première version on indique sous le champ `input` "champ obligatoire", si celui est rempli on efface l'avertissement.

1. Deuxième version, rien n'est affiché sous le champ obligatoire, si sélectionne le champ sans rien remplir et que l'on quitte ce champ un message s'affiche sous le champ en question.


Remarque pour la deuxième version utiliser l'événement `onBlur`
